# 💍 Casamento: Luiz & Isabela

Bem-vindo ao repositório do site oficial do casamento de **Luiz & Isabela**, que será realizado em **17 de julho de 2027**.

Este site foi construído utilizando o gerador de sites estáticos [Hugo](https://gohugo.io/) e baseado no tema [Gokarna](https://github.com/gokarna-theme/gokarna), mas foi extensivamente personalizado para refletir a identidade e o amor do casal com um visual **Premium e Minimalista**.

---

## ✨ Funcionalidades e Personalizações

- **Design Premium Bimodal:** Suporte nativo e otimizado para *Light Mode* (Claro) e *Dark Mode* (Escuro), com cores de destaque e contrastes ajustados dinamicamente para perfeita leitura em ambos os temas.
- **Monograma Exclusivo:** Um logo tipográfico "L&I" em formado SVG transparente presente na Navbar e no Favicon.
- **Contador Regressivo (Countdown):** Widget dinâmico na página principal marcando o tempo exato para o grande dia.
- **Nossa História (Timeline):** Uma linha do tempo visual contando a jornada do casal desde o primeiro encontro até o noivado.
- **Galeria Interativa (Masonry):** Layout estilo Pinterest fluido que se adapta a fotos horizontais e verticais. As imagens começam em *Preto e Branco* com um efeito elegante de transição para cor e Zoom (Lightbox) ao clicar.
- **Integração Spotify:** Playlist colaborativa incorporada ao site para que os convidados sugiram as músicas da festa.
- **O Casamento e RSVP:** Página detalhada com local do evento e explicações sobre a confirmação de presença.
- **Rotas Otimizadas:** Configuração de URLs relativas garantindo navegação sem falhas em navegadores móveis restritos (como o Safari no iOS).

---

## 🚀 Como Executar Localmente

Se você deseja rodar este projeto e visualizar as modificações na sua máquina:

1. **Pré-requisito:** Certifique-se de ter o [Hugo (versão Extended)](https://gohugo.io/installation/) instalado no seu computador.
2. Navegue até a pasta principal do site:
   ```bash
   cd exampleSite
   ```
3. Inicie o servidor local do Hugo:
   ```bash
   hugo server -D
   ```
4. Acesse pelo navegador: `http://localhost:1313`.

## 🌐 Deploy (Netlify)

O projeto está configurado para deploy automático super rápido via **Netlify**, e as diretrizes principais de Build estão declaradas no arquivo `netlify.toml` na raiz do repositório.
